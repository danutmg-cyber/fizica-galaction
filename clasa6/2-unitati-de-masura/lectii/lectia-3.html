Lecția 3 – Măsurarea directă a volumului

„Fizica nu este o materie dominată de formule, ci de logică”

Cuprins rapid

1. Ce este volumul? 2. Unități de măsură 3. Vase gradate 4. Corpuri solide

1. Ce este volumul?

Volumul arată cât spațiu ocupă un corp.

Orice lichid, gaz sau corp solid ocupă un anumit volum.

Volumul unui lichid se măsoară direct cu ajutorul unui vas gradat.

2. Unități de măsură pentru volum

  • mililitru – mL;
  • litru – L;
  • centimetru cub – cm³;
  • metru cub – m³.
Exemple utile: 1 L = 1000 mL, 1 cm³ = 1 mL.

3. Cum măsurăm volumul lichidelor

Folosim vase gradate: cilindru gradat, cană gradată, pipetă, balon gradat, flacon gradat.

Citirea se face la nivelul lichidului, privind corect gradația.

Vasul trebuie așezat pe o suprafață dreaptă, iar ochiul trebuie să fie la nivelul meniscului.

4. Cum măsurăm volumul unor corpuri solide

Pentru un corp solid neregulat putem folosi metoda deplasării apei.

  1. citim volumul inițial al apei;
  2. introducem corpul în vasul gradat;
  3. citim volumul final;
  4. scădem: Vcorp = Vfinal − Vinițial.
Metoda este utilă pentru pietre, bile metalice sau alte corpuri cu formă neregulată.

Reține

  • volumul arată cât spațiu ocupă un corp;
  • lichidele se măsoară direct cu vase gradate;
  • unități uzuale: mL, L, cm³, m³;
  • pentru corpuri neregulate putem folosi metoda deplasării apei.
