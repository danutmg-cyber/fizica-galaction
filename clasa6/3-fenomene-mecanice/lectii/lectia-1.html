Lecția 1 – Corp. Mobil. Reper. Sistem de referință

„Fizica nu este o materie dominată de formule, ci de logică”

Cuprins rapid

1. Corp și mobil 2. Ce este reperul? 3. Sistemul de referință 4. De ce este important?

1. Corp și mobil

În fizică, un corp este orice obiect din natură sau din jurul nostru.

Atunci când studiem mișcarea unui corp, îl putem numi mobil, adică obiectul a cărui poziție o urmărim în timp.

Un automobil, o minge, o carte sau chiar Pământul pot fi considerate corpuri fizice.

2. Ce este reperul?

Pentru a spune dacă un corp se mișcă sau stă pe loc, avem nevoie de un reper.

Reperul este corpul sau obiectul față de care descriem poziția altui corp.

De exemplu, un copac de pe marginea drumului poate fi reper pentru o mașină care trece pe lângă el.

3. Sistemul de referință

Un sistem de referință este format, în mod simplificat, din reperul ales și modul în care descriem poziția și timpul.

Cu ajutorul sistemului de referință putem spune unde se află corpul și cum își schimbă poziția.

4. De ce este important?

Aceeași mișcare poate fi descrisă diferit, în funcție de reperul ales.

De aceea, în fizică trebuie să precizăm mereu față de ce observăm corpul.

Fără reper, nu putem spune corect dacă un corp este în mișcare sau în repaus.

Reține

  • corpul este obiectul studiat în fizică;
  • mobilul este corpul a cărui mișcare este urmărită;
  • reperul este corpul față de care descriem poziția altui corp;
  • sistemul de referință ne ajută să descriem corect mișcarea.
