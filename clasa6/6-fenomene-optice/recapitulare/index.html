Recapitulare
Recitește ideile principale din capitol, apoi continuă cu testul final.
1. Lumina și sursele de lumină
  • Lumina este forma de energie care face posibilă vederea.
  • Sursele de lumină emit lumină proprie.
  • Există surse naturale, cum este Soarele, și surse artificiale, cum este becul.
  • Unele corpuri nu emit lumină, ci doar reflectă lumina primită.
2. Propagarea rectilinie a luminii
  • În medii omogene și transparente, lumina se propagă în linie dreaptă.
  • Raza luminoasă este reprezentarea direcției de propagare a luminii.
  • Această proprietate explică formarea umbrelor.
3. Umbra și penumbra
  • Umbra apare când un corp opac oprește complet lumina.
  • Penumbra apare când ajunge doar o parte din lumină.
  • Umbra este mai întunecată decât penumbra.
4. Reflexia luminii
  • Reflexia luminii este întoarcerea luminii dintr-o suprafață.
  • Fenomenul se observă mai bine pe suprafețe netede și lucioase.
  • Oglinda este un exemplu obișnuit de reflexie.
De reținut: multe fenomene optice din jurul nostru pot fi explicate prin propagarea rectilinie și reflexia luminii.
