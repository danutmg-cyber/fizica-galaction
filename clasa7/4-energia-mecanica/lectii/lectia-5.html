Lecția 5 – Energia potențială elastică

„Fizica nu este o materie dominată de formule, ci de logică.”

În această lecție înveți ce este energia potențială elastică, de ce depinde și cum se calculează.

energie elastică resort deformare stare de referință
Pagina 1 din 7

1. Ce observăm?

Un resort întins sau comprimat poate pune un corp în mișcare atunci când este lăsat liber.

Exemple

  • un arc comprimat într-o jucărie;
  • un resort alungit;
  • o praștie întinsă;
  • o trambulină deformată.

Concluzie

Un corp elastic deformat are energie datorită deformării sale.

Această energie se numește energie potențială elastică. Ea se definește pentru corpuri elastice deformate.

2. De ce depinde energia potențială elastică?

Energia potențială elastică depinde de:

  • constanta elastică a resortului, notată cu $k$;
  • deformarea resortului, notată cu $x$.

Dacă resortul este mai rigid

Pentru aceeași deformare, un resort cu constantă elastică mai mare are energie potențială elastică mai mare.

Dacă deformarea crește

Pentru același resort, o deformare mai mare înseamnă energie potențială elastică mai mare.

Deformarea apare la pătrat în formulă, deci influențează mult energia potențială elastică.

3. Formula energiei potențiale elastice

Energia potențială elastică a unui resort se calculează cu relația:

$$E_{pe} = \frac{kx^2}{2}$$

unde:

  • $E_{pe}$ este energia potențială elastică;
  • $k$ este constanta elastică a resortului;
  • $x$ este deformarea resortului.
Observă fracția:

Fracția este $\frac{kx^2}{2}$, adică produsul dintre constanta elastică și pătratul deformării, împărțit la 2.

4. Starea de referință

Energia potențială elastică se definește în raport cu o stare de referință.

Starea de referință

Starea de referință este starea nedeformată a corpului elastic.

Consecință

Dacă resortul nu este nici întins, nici comprimat, atunci $x = 0$ și energia potențială elastică este zero.

$$x = 0 \Rightarrow E_{pe} = 0$$
Un resort alungit cu 2 cm și același resort comprimat cu 2 cm au aceeași energie potențială elastică, deoarece în formulă apare $x^2$.

5. Exemplu de calcul

Un resort are constanta elastică $k = 100\,N/m$ și este deformat cu $x = 0{,}2\,m$.

Aplicăm formula:

$$E_{pe} = \frac{kx^2}{2} = \frac{100 \cdot 0{,}2^2}{2} = \frac{100 \cdot 0{,}04}{2} = \frac{4}{2} = 2\,J$$
Rezultatul este $E_{pe} = 2\,J$.

6. Reținem

  • energia potențială elastică este energia datorată deformării unui corp elastic;
  • formula este $E_{pe} = \frac{kx^2}{2}$;
  • energia depinde de constanta elastică $k$ și de deformarea $x$;
  • unitatea de măsură este joule (J);
  • în starea nedeformată, $x = 0$ și energia potențială elastică este zero.
Alege o singură variantă corectă.

Verificare rapidă

Bifează enunțul corect.
